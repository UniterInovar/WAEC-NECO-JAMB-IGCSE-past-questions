# Past Questions & Answers App

A comprehensive tool for students and teachers to access past questions for WAEC, NECO, NABTEB, JAMB, and IGCSE.

## Structure
- `/backend`: FastAPI server and SQLite database.
- `/frontend`: Modern dark-mode dashboard.
- `/scrapers`: Python-based scraping tools for web and PDF.
